# FoodieQR Ordering UI v2 (Static)

This is a production-friendly static demo using only HTML/CSS/JS.

## Deploy
Upload all files to any static host (Netlify/Vercel/GitHub Pages/Cloudflare Pages).

**Important:** Camera-based QR scanning usually requires HTTPS on mobile browsers.

## QR Format
Encode a URL like:
hotel.html?hotel=hotel-a&table=02

## Realtime Demo
Open 2 tabs:
- kitchen.html?hotel=hotel-a
- track.html
Status updates sync via localStorage.
